Developing with the Centrify MAS API
This guide shows you how to quickly start development with the Centrify Mobile Authentication Service (MAS) API to create a mobile application designed to run on an Android or iOS mobile device.
The Centrify MAS API is specifically designed to provide single-sign-on (SSO) capabilities to applications runningon an Android or iOS mobile device. The Centrify application on the device authenticates the user through Centrify Cloud Services and, after that, the user does not need to provide credentials in order to use any of the SSO applications.
Refer to Centrify Mobile Authentication Services SDK Implementation Guide to understand the roles of the MAS API, Centrify Cloud Service, the Centrify application, other mobile applications, and SaaS applications.
