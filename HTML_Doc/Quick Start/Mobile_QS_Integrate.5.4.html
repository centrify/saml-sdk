Install and configure the Centrify application
The Centrify application can be run on any mobile device running Android 2.3 or later or iOS 6 or later. The Centrify application must be version 13.10 or later.
1
Download the Centrify application from Google Play or the Apple App Store and install it on the mobile device.
2
Launch the Centrify app and, under Settings, make sure the login URL is set to https://cloud.centrify.com. Click Enable Debug Mode.
Figure 1. Centrify app settings
 
3
On the Enrollment screen, enter the Active Directory credentials for the user allowed to register the mobile device with Centrify Cloud Service.
