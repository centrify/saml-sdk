Integration of Android MAS Apps : Install and run the demo application
Install and run the demo application
The demo applications can be tested on any mobile device running Android 2.3 or later.
1
Install the GenericSSODemo1.apk sample application on the Android device.
Note Android program (apk) files can be installed on both devices and emulators using adb tools, which are part of the Android SDK. For more details, refer to http://developer.android.com/sdk/index.html.
2
Launch the sample application.
3
Try the buttons in the sample application to see the information returned by Centrify Cloud Service.
4
Close the sample application and reopen it. You should not have to enter credentials again.
