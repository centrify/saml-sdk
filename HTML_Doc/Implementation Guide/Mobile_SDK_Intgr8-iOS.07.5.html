Integration of iOS MAS Apps : Install and run the demo app
Install and run the demo app
The demo apps can be tested on any mobile device running iOS 6.0 or later.
1
Install the CentrifySDKSample sample app on the iOS device or use the emulator in Xcode.
Note For details on how to test an iOS app on a device, refer to https://developer.apple.com/library/mac/documentation/IDEs/Conceptual/AppDistributionGuide/TestingYouriOSApp/TestingYouriOSApp.html.
2
Launch the sample app.
3
Try the buttons in the sample app to see the information returned by Centrify Cloud Service.
4
Close the sample app and reopen it. You should not have to enter credentials again.
